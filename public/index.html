About
What is UCP?

The Universal Commerce Protocol (UCP) is an open standard for agentic and headless commerce. It enables AI agents, publishers, and any digital surface to discover products, create checkout sessions, and complete purchases — all through a unified API.

UCP defines a standard discovery profile, catalog search, checkout lifecycle (create, update, complete, cancel), fulfillment handling, payment processing via Google Pay, and order management. Any commerce platform can implement UCP to make their merchants' products purchasable from anywhere.

The protocol is maintained by Google's UCP team and published at ucp.dev.

Versions
Supported Spec Versions
This tool can test against any published version of the UCP specification. Select the version when running tests, or use auto-detect to test against the server's reported version.

2026-04-08

Released April 9, 2026

Current and latest. Adds catalog search/lookup, cart capability, request/response signing, eligibility claims, formalized totals, variant.options rename, embedded checkout protocol. Major release with breaking changes from 2026-01-23.

2026-01-23

Released January 23, 2026

Added fulfillment groups, shipping option selection, discount capability, buyer consent, order webhooks. The official Google conformance suite currently targets this version.

2026-01-11

Released January 11, 2026

Initial public release. Core checkout lifecycle: discovery, create, update (PUT), complete, cancel. Google Pay payment handler. Basic fulfillment with shipping address.
